# Security Policy

## 支持的版本

本仓库为分类体系仓库，不直接包含可执行代码。安全更新主要针对仓库基础设施。

## 报告安全漏洞

如果你发现了安全漏洞，请**不要**通过公开 Issue 报告。

请通过以下方式私下报告：

- 发送邮件至：[security@hermes-agent.example.com](mailto:security@hermes-agent.example.com)
- 主题格式：`[SECURITY] digital-employee-taxonomy - <简要描述>`

我们将在 48 小时内确认收到报告，并在 7 天内提供初步评估。

## 安全最佳实践

使用本仓库的分类体系时，请遵循：

- 定期审查和更新引用的分类版本
- 关注原仓库的安全公告
- 在隔离环境中测试第三方连接器
